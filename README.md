High-performance Network Mapper for discovery and management

## CLI (rmap)
- [rushmap](https://github.com/shellrow/rushmap/tree/main/rushmap)
## GUI (RushMap Desktop)

## Features
- Port Scan
    - Service detection
    - OS detection
- Host Scan
- Ping
- Traceroute
- Subdomain scan

- Linux
